Сервер удаленного доступа к базе данных SQL SERVER на WCF 
Суть задания состоит в следующем: клиент посылает запросы типа SELECT к серверу, на котором установлена служба, и служба обрабатывает этот запрос и возвращает клиенту результат. 
Базу данных можно использовать любую (но СУБД SQL SERVER), только необходимо в конфиге прописать нужные параметры доступа к базе.
